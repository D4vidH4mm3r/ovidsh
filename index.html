Ovidsh (fork)

Bookmarklets to assist in saving search histories

View project on GitHub

This is a fork of ovidsh by GitHub user ipoga.

The bookmarklets are intended to help make search histories (from Ovid or Cinahl so far) easier to extract. In general, they extract the search history information and display it as raw text which can easily be copied into a table in some office program (for instance Microsoft Office Word). The raw text is displayed in a textarea and should be automatically selected by clicking the textarea.

In order to use the bookmarklets, you can simply save them as bookmarks. Opening them will execute the code on the page currently loaded in the browser (which should be a search history). In most browsers, you can add the bookmarklets to the bookmarks bar by simply dragging them from this page.

Ovid

The bookmarklet assumes that you are viewing a fully expanded search history. The bookmarklet is here: OvidSH.

Cinahl (or possibly other publications in EBSCOhost)

The Cinahl bookmarklet assumes that you are viewing an updated search history. The bookmarklet is here: CinahlSH.
