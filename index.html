Ovidsh (fork)

Bookmarklet to assist in saving search histories

View project on GitHub

This is a fork of ovidsh by GitHub user ipoga. So far, there are bookmarklets for Ovid and Cinahl (still a bit rough).

Ovid

Save the bookmarklet from http://ipoga.github.io/ovidsh to generate a machine- readable search history from the OvidSP search interface.

Cinahl

Adding and using the bookmarklet is more or less the same as for the Ovid bookmarklet by ipoga. The Cinahl bookmarklet assumes that you are viewing an updated search history. When clicked, it will add a text block above the search history with an easy-to-copy version of the search history. Currently, this block stays until you reload the page / navigate away. The bookmarklet is here: Cinahl.
