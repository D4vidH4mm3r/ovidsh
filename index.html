Ovidsh (fork)

OvidSP search history saving bookmarklet

View project on GitHub

This is simply a fork of ovisdh by ipoga.

Save the bookmarklet from http://ipoga.github.io/ovidsh to generate a machine- readable search history from the OvidSP search interface.

Soon, there will be a similar bookmarklet for the Cinahl search interface (I just need to get a functioning URL for the js).
